```python                                                                       
# @module: src/paper_trading/integration.py                                     
# @deps: src/orchestrator/qroe_engine.py, src/core/coherence/logodds_fusion.py, 
 src/telemetry/logger.py, src/paper_trading/engine.py                           
# @status: IMPLEMENTED                                                          
# @last_update: [timestamp]                                                     
# @summary: Async wiring layer connecting paper engine to QROE decision loop,   
coherence fusion, telemetry, and validation gates.                              
                                                                                
class PaperIntegrationOrchestrator:                                             
    def __init__(self, qroe: QROEEngine, telemetry: TelemetryMatrix,            
paper_engine: PaperTradingEngine):                                              
        self.qroe = qroe                                                        
        self.telemetry = telemetry                                              
        self.paper_engine = paper_engine                                        
        self.trace_counter = 0                                                  
                                                                                
    async def run_cycle(self, market_tick: MarketTick) -> Dict:                 
        trace_id = f"PAPER-{self.trace_counter:06d}"                            
        self.trace_counter += 1                                                 
                                                                                
        # 1. QROE selects phase & profile                                       
        phase_decision = await self.qroe.select_phase(market_tick, trace_id)    
                                                                                
        # 2. Coherence fusion (simulated agent outputs for validation)          
        coherence_input = CoherenceInput(                                       
            agent_logodds=phase_decision['simulated_signals'],                  
            agent_weights=phase_decision['simulated_weights'],                  
            prior_logodds=0.0,                                                  
            regime_penalty=phase_decision['regime_penalty']                     
        )                                                                       
        L_total = fuse_coherence(coherence_input)                               
                                                                                
        # 3. Route to paper execution                                           
        exec_result = await self.paper_engine.simulate_cycle(trace_id, L_total, 
 market_tick)                                                                   
                                                                                
        # 4. Telemetry emission                                                 
        await self.telemetry.record(TelemetryEvent(                             
            trace_id=trace_id,                                                  
            level="INFO",                                                       
            component="paper_integration",                                      
            event_type="cycle_complete",                                        
            context={"L_total": L_total, "phase": phase_decision['phase'],      
"filled": exec_result['filled']},                                               
            metrics={"latency_us": exec_result['latency_us'], "tce_bps":        
exec_result['tce_bps']}                                                         
        ))                                                                      
                                                                                
        return {"trace_id": trace_id, "decision": phase_decision, "execution":  
exec_result}                                                                    
```

## ⚙️ REGRAS DE IMPLEMENTAÇÃO (INVARIÁVEIS)                                     
1. **Async-First & Non-Blocking**: Todo I/O (CCXT, risk check, telemetry,       
reconciliation) via `asyncio`. Zero chamadas síncronas no hot path. Latência    
decision→submit < 5ms p99.                                                      
2. **Idempotência por Construção**: Cada ordem usa `nonce` monotônico por       
sessão. Re-submissão com mesmo nonce retorna estado existente, nunca duplica.   
Cache de nonces com TTL=300s.                                                   
3. **Pre-Flight Risk Gates HARD BLOCK**: Se QUALQUER gate falhar (`margin <     
required`, `DD > limit`, `vol > cap`, `killswitch_active`), ordem é REJEITADA   
com `AssertionError` + log CRITICAL. Zero override sem autenticação CEO + 2FA   
simulada.                                                                       
4. **Reconciliação Contínua**: Loop assíncrono a cada 5-10s compara estado      
interno vs exchange REST. Drift > 0.01% → auto-correção (cancel residual ou     
adjust size) + `CRITICAL` alert.                                                
5. **Kill-Switch Atômico**: Latência < 500ms do trigger ao flatten completo.    
Executa: cancel-all → market flatten (com slippage cap) → state freeze →        
telemetry burst. Irreversível sem restart manual.                               
6. **Modos de Operação**:                                                       
   - `--mode dry-run`: Simula gateway sem chamar exchange (valida contracts,    
risk gates, reconciliation logic)                                               
   - `--mode shadow`: Envia ordens reais mas com sizing 0.001 (valida slippage, 
 latency, fee structure)                                                        
   - `--mode live`: Execução real com gates ativos e monitoring total           
7. **PowerShell Compatibility**: Zero Unix-only commands. Scripts `.ps1` com    
`param()`, `Test-Path`, `Select-String`, `Invoke-RestMethod`.                   
8. **Headers Obrigatórios**: `# @module`, `# @deps`, `# @status`, `#            
@last_update`, `# @summary` em TODOS os arquivos.                               
9. **Zero Placeholders**: Código completo, testado, integrado. Nada de `#       
TODO`, `pass`, ou `will_implement_later`.                                       
10. **Testes Property-Based**: Hypothesis deve provar que `∀orders: idempotent  
∧ risk_gated ∧ no_overexposure ∧ killswitch_atomic`.
